Woocommerce M.A.P. allows you to set a Minimum Advertised Price for each product

== Description ==
NOTE: You must install the main "WooCommerce - excelling eCommerce" plugin in order to use this plugin

Woocommerce M.A.P. adds a Minimum Advertised Price field to the product edit admin.  If the product price is lower than the M.A.P. then the M.A.P. price is displayed instead.  On the product frontend page users can "Click to View Our Price" in order to be shown the real price.

When products are added to cart they will be charged at the real price, the M.A.P. price is for display only in order to meet certain manufacturers requirements.

== Installation ==

1. Unzip/Extract the plugin zip to your computer
2. Upload `\woocommerce-mapprice\` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add/Edit a product and enter a M.A.P. Price
